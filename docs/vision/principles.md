# Principles

## Product Principles

1. GUI-first operation for core systems.
2. Command parity for all GUI actions.
3. Safe defaults over permissive shortcuts.
4. Clear permission boundaries.
5. Consistent response language and feedback.

## Engineering Principles

1. Docs contracts lead implementation.
2. Small cohesive files and modules.
3. Deterministic verification via compose.
4. No silent failures on command execution.
5. Explicit thread-safety for Folia contexts.
