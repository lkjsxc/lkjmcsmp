# Menu Tree

## Inventory Size Contract

- All plugin menus use 54 slots (large chest layout).
- Shared navigation row reserves `Refresh` and `Back` controls.

## Root Menu (`/menu`)

1. Teleport
2. Homes
3. Warps
4. Team
5. Points Shop
6. Progression
7. Settings

## Teleport Menu

- Random Teleport
- TPA Request (target picker)
- TPA Here (target picker)
- Pending Request Accept/Deny
- Direct Player Teleport (permission-gated)

## Homes Menu

- Home list
- Left-click home entry: teleport
- Right-click home entry: delete
- Set default `home`
- Delete default `home`

## Warps Menu

- Warp list
- Teleport to warp
- Manage warps (permission-gated)

## Team Menu

- Team info
- Create team
- Invite player (target picker)
- Accept invite / leave / disband
- Team home and sethome

## Points Shop Menu

- Convert cobblestone
- Buy logs
- Buy dirt
- Exchange history

## Progression Menu

- Milestone list
- Completed milestones
- Reward claim actions
