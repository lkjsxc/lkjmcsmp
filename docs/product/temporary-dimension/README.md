# Temporary Dimension Contracts

## Summary

Players purchase isolated, time-limited dimension instances with Cobblestone Points. Each instance is an independent world with configurable environment (End, Nether, or Overworld).

## Rules

1. Each purchase creates an independent, isolated dimension instance.
2. Multiple instances may exist simultaneously.
3. Instances have a strict lifecycle with deterministic cleanup.
4. Player safety on expiry is mandatory; no stranding in deleted dimensions.
5. Environment type is determined at purchase time via shop entry configuration.
