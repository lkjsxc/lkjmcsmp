# Repository Contracts

## Goal

Define repository structure, hard constraints, and change workflow rules.

## Rules

1. Layout contracts guide where new code and docs live.
2. Rule contracts define line limits and docs-first enforcement.
3. Workflow contracts define batch order and commit cadence.

- [layout/README.md](layout/README.md): root and source layout contracts
- [rules/README.md](rules/README.md): docs-first and line-limit constraints
- [workflow/README.md](workflow/README.md): change sequencing and commit policy
